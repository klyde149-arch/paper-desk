# trading-sim / paper-desk — автономная paper-trading система

Симуляция торговли криптой (и исследования рынка РФ). **Только виртуальные деньги, публичные API без ключей.** Начато 2026-07-08. **С 2026-07-10 торгует автономный бот 24/7** — GitHub Actions запускает тик каждые ~15 минут независимо от локального компьютера.

## Автоматика (главное)

- **Движок:** `tools\auto_trade.ps1` (+ `tools\lib_engine.ps1`). Каждый тик: реплей закрытых 1m-свечей с прошлого тика — **стопы/TP1 по касанию цены** (gap-aware), фандинг на 8h-слотах по реальным ставкам, трейл раннера по закрытию 4h за EMA20, лимиты риска, роллы UTC-дня; на свежезакрытом 4h-баре — сканер v2 и автовход; персист + журнал; пересборка дашборда.
- **Рынок РФ (`tools\rf_engine.ps1`, с 2026-07-10):** живой форвард-тест портфелей **C2 и C3b** (два счёта по $10k: Donchian-ядро + setup A + momentum-акции) — решения по дневным барам MOEX ISS, входы по открытию следующей сессии, стопы по касанию (часовики), платные роллы фронт-контрактов. Состояние в `data\rf\`, живой блок — вкладка «Фьючерсы РФ» дашборда.
- **Расписание:** `.github/workflows/tick.yml` — cron `*/15`, а также ручной запуск и запуск на каждый push. Раннер Windows, PS 5.1. Bybit заблокирован с раннеров (403) — слой данных автоматически падает на зеркало/BingX.
- **Состояние в git:** бот коммитит `portfolio.json`, `journal.md`, `data\live_*.json`, `challenge\portfolio.json` в main после каждого тика с изменениями. Сайт публикуется тем же workflow (Pages artifact = `report\`).
- **Килл-свитч:** закоммитить пустой файл `data\HALT` в main — тики выходят сразу, позиции не трогаются. Убрать файл — бот продолжит с догоном пропущенного.
- **Комиссии:** Bybit деривативы, тейкер 0.055% на сторону (включая TP1), слиппедж 0.03% (стопы +0.05%).
- **Правила сосуществования:** локальная сессия (Claude/пользователь) начинается с `git pull`, заканчивается push (`tools\deploy_site.ps1`). Локальные тики руками не запускать, пока работает облако (двух писателей быть не должно); для отладки — `-DryRun` или сначала `data\HALT`.

## Цикл (автоматизирован с 2026-07-10)

Прежний ручной цикл агента упразднён — всё делает `tools\auto_trade.ps1` в облаке (см. «Автоматика»). Роль агент-сессий теперь: анализ, исследования, доработки. Порядок сессии: `git pull` → читать хвост `journal.md` + `portfolio.json` → работать → push.

Локальный терминал: `tools\start_terminal.ps1 -Launch` → http://localhost:8377/ (или публичный https://klyde149-arch.github.io/paper-desk/)

Подробности и договорённости — в памяти Claude (`trading-sim-project.md`) и в `docs\strategy.md`.
