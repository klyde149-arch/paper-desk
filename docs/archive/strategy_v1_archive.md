# Торговый регламент — paper trading (виртуальный счёт $10 000)

**Статус:** активен с 2026-07-09. Реальные деньги НЕ используются. Все сделки виртуальные, цены берутся с публичных API BingX/Bybit в момент решения.

**Цель (рекалибрована по бэктесту 16.4 мес, см. [backtest_report.md](backtest_report.md)):** реалистичное ожидание этой стратегии — **2–4%/мес при просадках 12–25%**. Заявленные 5–10%/мес недостижимы без риска просадки ~30%, а лимит просадки ≤8% недостижим без падения доходности до ~1%/мес — эти условия математически в конфликте. **Приоритет №1 — сохранение капитала.** Рабочий режим ниже настроен на баланс, а не на максимум доходности.

**Рынки:** только крипта (BingX perpetual). Российские акции (MOEX) протестированы 2026-07-09 и **отвергнуты**: дневки −2.2%/3 года (PF 0.94), часовики −68.5%/2 года — издержки съедают ~65% размера стопа против ~10% у крипты. Подробности в [backtest_report.md](backtest_report.md).

## 1. Вотчлист

BTC, ETH, SOL, BNB, XRP, ADA, AVAX, LINK (бессрочные контракты USDT на BingX). **DOGE исключён** — по бэктесту стабильно убыточен (мем-волатильность ломает трендовую логику). Другие ликвидные пары — только при исключительном сетапе.

## 2. Определение режима рынка (4h + 1D по BTC и по активу)

- **Аптренд:** цена выше EMA50(4h), EMA20 > EMA50, растущие максимумы/минимумы.
- **Даунтренд:** зеркально.
- **Флэт:** цена пилит EMA20/50, диапазон очерчен, EMA плоские.

## 3. Сетапы

### A. Трендовый откат (основной)
- **Фильтр тренда BTC (обязателен, подтверждён бэктестом — PF 1.21→1.45):** не открывать лонги, если BTC в даунтренде 4h (цена < EMA50 и EMA20 < EMA50); не открывать шорты, если BTC в аптренде 4h. Не идти против главного рынка.
- **ATR-кэп (принят по эксперименту №5, 2026-07-08):** не входить, если ATR(14, 4h) > 3% от цены — рынок перегрет, стопы неоправданно широкие. Улучшает все метрики (PF 1.45→1.48, DD 24.6→23.4%), в walk-forward нигде не вредит. Обратный фильтр (мин. ATR) проверен и ВРЕДЕН — спокойные откаты не отсекать.
- Направление — только по тренду 4h, подтверждённому 1D-структурой.
- Вход: откат к зоне EMA20–EMA50 (4h), RSI(4h) откатился к 40–50 в аптренде (50–60 в даунтренде для шорта), появилось свечное подтверждение (поглощение/пин-бар/возврат выше уровня).
- Стоп: за локальный экстремум отката, минимум 1×ATR-эквивалент (~ размах последних 3–5 свечей 4h).
- Тейк: TP1 = 1.5R (закрыть 50%), остаток — трейлинг за EMA20(4h) или структурные уровни. Минимальное соотношение R:R для входа — 1:1.5 до TP1.

### B. Флэтовый отбой — ПРИОСТАНОВЛЕН (бэктест 2026-07-08)
Механическая проверка (вход от границ 30-барного диапазона при RSI<35/>65 со свечным подтверждением): 9 сделок за 16 мес, win-rate 22%, −$480. В текущем виде сетап убыточен — не использовать, пока не найдены параметры с положительным ожиданием. Торгуем только сетап A.

### C. Сентимент-фильтры (модификаторы, не самостоятельные сигналы)
- **Фандинг** (Bybit): > +0.05%/8ч — лонги перегреты, новые лонги запрещены; < −0.05%/8ч — зеркально для шортов. Экстремальный фандинг против моей позиции — довод к частичной фиксации.
- **Fear & Greed:** ≤20 — запрещены новые шорты по маячащему «дну» (шорт только при явном сломе структуры); приоритет — лонговые сетапы с подтверждением разворота. ≥80 — зеркально.
- **Резкое движение >3%/час** — сначала WebSearch новостей, потом решение.

## 4. Риск-менеджмент (нарушать НЕЛЬЗЯ)

| Правило | Значение |
|---|---|
| Риск на сделку | **0.5–0.75%** от депозита (рекалибровано; 1% давало просадку ~25%) |
| Размер позиции | = риск$ / (расстояние до стопа в %) |
| Плечо | ≤5x (плечо — следствие размера позиции, не цель) |
| Одновременных позиций | **≤2** (по бэктесту 2 лучше держат просадку, чем 3) |
| Корреляция | BTC+ETH+альты ходят вместе — max 2 однонаправленные крипто-позиции |
| Дневной лимит убытка | −3% от эквити → стоп-торговля до следующего дня UTC |
| Мягкая просадка | −8% от пика → снизить риск вдвое, пересмотреть сетапы |
| Жёсткая просадка | −15% от пика → полная остановка, разбор стратегии |
| Стоп-лосс | обязателен ДО входа, двигать можно только в сторону прибыли |

## 5. Модель издержек (честная симуляция)

- Комиссия: 0.05% тейкер на вход и выход (BingX перпы).
- Проскальзывание: 0.03% на рыночные входы/выходы, стопы исполняются ПО стоп-цене + 0.05% проскальзывания (консервативно).
- Фандинг: каждые 8ч (00/08/16 UTC) по фактической ставке Bybit, начисляется на номинал позиции.
- Исполнение стопов/тейков между проверками: по high/low свечей 1h с момента прошлой проверки. Если свеча задела и стоп и тейк — считается, что сработал СТОП (консервативно).

## 6. Дисциплина цикла

Каждое пробуждение: (1) обновить цены и P&L открытых позиций, проверить стопы/тейки по свечам; (2) начислить фандинг; (3) проверить лимиты риска; (4) искать новые сетапы; (5) записать всё в journal.md и portfolio.json; (6) запланировать следующее пробуждение (30–60 мин; чаще при позиции у стопа/тейка).

Запрещено: входить без записанного заранее стопа и тейка; усредняться против позиции; «отыгрываться» после стопа; менять правила на ходу без записи в журнал с обоснованием.
