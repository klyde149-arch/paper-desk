# Торговый регламент v2 «комбо» — paper trading (виртуальный счёт $10 000)

**Статус:** v2 активен с 2026-07-10 (решение пользователя «внедряй v2 комбо»; v1 действовал 2026-07-09 → 2026-07-10, архив: [archive/strategy_v1_archive.md](archive/strategy_v1_archive.md), обоснование v2: [backtest_v2_report.md](backtest_v2_report.md)). Реальные деньги НЕ используются. Все сделки виртуальные, цены — публичные API Bybit (фолбэк BingX) в момент решения.

**Исполнение автоматизировано:** цикл ведёт `tools\auto_trade.ps1` (тик каждые ~15 мин в GitHub Actions, 24/7 независимо от ноутбука). Стопы и тейки срабатывают **по касанию цены** (проверка по 1-минутным свечам, при простое — реплей пропущенных минут). Агент (Claude) занимается только анализом и доработками, не исполнением.

**Цель (рекалибрована по длинной истории 6.2 г, см. [backtest_v2_report.md](backtest_v2_report.md) §5):** в трендовых фазах — **1.5–2.5%/мес**; на полном цикле (включая вязкие боковики) — **0.7–1%/мес при DD 13–15%**; в затяжном флэте около нуля месяцами — норма, а не поломка. **Приоритет №1 — сохранение капитала.**

**Рынки:** только крипта (Bybit linear perpetual; расхождение цен с BingX в среднем 0.006% — биржи взаимозаменяемы). Российские акции (MOEX) протестированы 2026-07-09 и **отвергнуты**: дневки −2.2%/3 года (PF 0.94), часовики −68.5%/2 года — издержки съедают ~65% размера стопа против ~10% у крипты. Подробности в [backtest_report.md](backtest_report.md).

## 1. Вотчлист

BTC, ETH, SOL, BNB, XRP, ADA, AVAX, LINK (бессрочные контракты USDT, Bybit). **DOGE исключён** — по бэктесту стабильно убыточен (мем-волатильность ломает трендовую логику). Другие ликвидные пары — только при исключительном сетапе.

## 2. Определение режима рынка (4h + 1D по BTC и по активу)

- **Аптренд:** цена выше EMA50(4h), EMA20 > EMA50, растущие максимумы/минимумы.
- **Даунтренд:** зеркально.
- **Флэт:** цена пилит EMA20/50, диапазон очерчен, EMA плоские.

## 3. Сетапы

### A. Трендовый откат (основной)
- **Фильтр тренда BTC (обязателен, подтверждён бэктестом — PF 1.21→1.45):** не открывать лонги, если BTC в даунтренде 4h (цена < EMA50 и EMA20 < EMA50); не открывать шорты, если BTC в аптренде 4h. Не идти против главного рынка.
- **FlatMode = skip (v2, обязателен):** если BTC на 4h во флэте (режим 'range' — цена пилит EMA50, EMA20 ≈ EMA50), **все новые входы запрещены** по любым парам. Открытые позиции ведутся по обычным правилам. Обоснование: весь чистый минус стратегии создаёт BTC-флэт (PF 0.56); skip лучше halfrisk и по итогу, и в вязком периоде (PF 1.08 vs 1.02).
- **ATR-кэп (принят по эксперименту №5, 2026-07-08):** не входить, если ATR(14, 4h) > 3% от цены — рынок перегрет, стопы неоправданно широкие. Улучшает все метрики (PF 1.45→1.48, DD 24.6→23.4%), в walk-forward нигде не вредит. Обратный фильтр (мин. ATR) проверен и ВРЕДЕН — спокойные откаты не отсекать.
- Направление — только по тренду 4h, подтверждённому 1D-структурой.
- Вход: откат к зоне EMA20–EMA50 (4h), RSI(4h) откатился к 40–50 в аптренде (50–60 в даунтренде для шорта), появилось свечное подтверждение (поглощение/пин-бар/возврат выше уровня).
- Стоп: за локальный экстремум отката, минимум 1×ATR-эквивалент (~ размах последних 3–5 свечей 4h).
- Тейк: TP1 = 1.5R (закрыть 50%), остаток — трейлинг за EMA20(4h) или структурные уровни. Минимальное соотношение R:R для входа — 1:1.5 до TP1.

### B. Флэтовый отбой — ПРИОСТАНОВЛЕН (бэктест 2026-07-08)
Механическая проверка (вход от границ 30-барного диапазона при RSI<35/>65 со свечным подтверждением): 9 сделок за 16 мес, win-rate 22%, −$480. В текущем виде сетап убыточен — не использовать, пока не найдены параметры с положительным ожиданием. Торгуем только сетап A.

### C. Сентимент-фильтры
- **Фандинг (Bybit) — ЖЁСТКОЕ ворото входа (v2, подтверждено данными: +27пп за 6.2 г):** > +0.05%/8ч — новые лонги запрещены; < −0.05%/8ч — новые шорты запрещены. Экстремальный фандинг против открытой позиции — довод к частичной фиксации.
- **Fear & Greed:** ≤20 — запрещены новые шорты по маячащему «дну» (шорт только при явном сломе структуры); приоритет — лонговые сетапы с подтверждением разворота. ≥80 — зеркально.
- **Резкое движение >3%/час** — сначала WebSearch новостей, потом решение.

## 4. Риск-менеджмент (нарушать НЕЛЬЗЯ)

| Правило | Значение |
|---|---|
| Риск на сделку | **1%** от эквити (решение пользователя 2026-07-10: самый прибыльный профиль по матрице риска — PF 1.38, ~1.9%/мес, историческая просадка ~34%; консервативная альтернатива 0.6%/2поз задокументирована в журнале №33) |
| Размер позиции | = риск$ / (расстояние до стопа в %) |
| Плечо | ≤5x (плечо — следствие размера позиции, не цель) |
| Одновременных позиций | **≤3** (3-й слот улучшает PF 1.29→1.38 — подбирает сигналы, которые терялись) |
| Корреляция | покрыта лимитами просадки; все 3 позиции могут быть однонаправленными |
| Дневной лимит убытка | −5% от эквити → стоп новых входов до следующего дня UTC |
| Мягкая просадка | −16% от пика → риск вдвое (0.5%), возврат при <12% |
| Жёсткая просадка | −35% от пика → закрыть всё, полная остановка до решения пользователя |
| Стоп-лосс | обязателен ДО входа, двигать можно только в сторону прибыли |

Ожидание при этом профиле (6.2 г, комиссии Bybit): полный цикл ~1.9%/мес геом. (+324%), тренды 3–5%/мес, вязкие боковики около нуля; просадки до трети счёта — ЗАПЛАНИРОВАННАЯ цена доходности, не поломка. Схема выхода TP1 50% @1.5R → БУ → раннер за EMA20 проверена против одиночного тейка (PF 1.01–1.03 — отвергнут) и чистого трейла (DD ×2, минус в боковике — отвергнут).

## 5. Модель издержек (честная симуляция, комиссии Bybit — с 2026-07-10)

- Комиссия: **0.055% тейкер на вход и на каждый выход** (реальный тейкер Bybit деривативы, non-VIP). TP1-лимит тоже считаем по тейкеру — консервативно. (До 2026-07-10 использовалось 0.05%; сделка T1 учтена по старой модели.)
- Проскальзывание: 0.03% на рыночные входы/выходы, стопы исполняются ПО стоп-цене + 0.05% проскальзывания (консервативно).
- Фандинг: каждые 8ч (00/08/16 UTC) по фактической ставке Bybit, начисляется на номинал позиции (шорт при положительной ставке ПОЛУЧАЕТ).
- **Исполнение стопов/тейков: по КАСАНИЮ цены** — движок проверяет каждую 1-минутную свечу (при простое реплеит пропущенные минуты хронологически). Гэп за уровень — исполнение по открытию свечи (хуже уровня). Если одна минутная свеча задела и стоп и тейк — сработал СТОП (консервативно).

## 6. Дисциплина цикла (автоматизирована с 2026-07-10)

Тик `tools\auto_trade.ps1` каждые ~15 мин (GitHub Actions): (1) реплей минутных свечей с прошлого тика — стопы/TP1/фандинг/MFE-MAE; (2) трейл раннера по закрытию 4h за EMA20; (3) лимиты риска (−3% день → стоп входов до следующего UTC-дня; −8% от пика → риск ×0.5; −15% → закрыть всё и полная остановка до решения пользователя); (4) на свежезакрытом 4h-баре — сканер v2 (`scan_signals.ps1`) и автоvход при прохождении всех ворот (карточка тезиса генерируется из данных сканера); (5) запись portfolio.json / live_trades.json / journal.md (события и суточная сводка); (6) пересборка vizdata и публикация дашборда.

Ручное вмешательство: файл `data\HALT` в репозитории мгновенно останавливает тики (килл-свитч). Агент-сессии сначала делают `git pull`, работают, пушат.

Запрещено: входить без записанного заранее стопа и тейка; усредняться против позиции; «отыгрываться» после стопа; менять правила на ходу без записи в журнал с обоснованием.
